# Legal Case Tracker ⚖️

A full-stack legal management workspace designed for legal professionals to streamline operations, manage case lifecycle states, and track real-time litigation metrics.

## 🚀 Live Links
* **Frontend Web Application:** [Live on Vercel](https://legal-case-manager-li4r.vercel.app)
* **Backend API Service:** [Hosted on Render](https://lextrack-backend-8rih.onrender.com)

---

## ✨ Features
* **Practice Overview Dashboard:** Real-time analytics showcasing total cases, active vs. disposed ratios, and legal category tracking (Civil, Criminal, Revenue).
* **Active Docket Spotlight:** Comprehensive data grid tracking case numbers, matters, clients, and real-time status badges.
* **Integrated Legal Portals:** Quick access to essential legal research services including eCourts, IndianKanoon, and Vakilsearch.
* **JWT Authentication Support:** Pre-configured architecture for secure API endpoints and token-based authentication protocols.
* **Dark Mode Capability:** Built-in UI flexibility allowing users to toggle visual themes for comfort.

---

## 🛠️ Tech Stack
* **Frontend:** React, Vite, Tailwind CSS
* **Backend:** Node.js, Express (or Python/Flask/Django depending on your exact backend configuration)
* **Hosting Pipelines:** Vercel (Frontend), Render (Backend API)
